# React Redux Boilerplate

### Usage

```
npm install
npm start
open http://localhost:8080
```

## Dependencies

1. React
1. Redux
1. redux-logger
1. redux-devtools (via chrome extension)
1. Webpack
1. webpack-dev-server
1. babel-loader
